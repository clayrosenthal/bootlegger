# Bootlegger

FastAPI Server for [moonshine](https://github.com/moonshine-ai/moonshine) that serves OpenAI compatible API

## Moonshine

A speech-to-text (stt) model and engine with a python API. Uses ONNX to run on various hardware setups.


## OpenAI API

Read the [full spec](./OPENAI_AUDIO_API.md) for more details, but this project is focusing on transcriptions.
